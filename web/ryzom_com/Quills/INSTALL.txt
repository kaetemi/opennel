Installation Instructions
-------------------------

This is a very prelimanary set of installation instructions.  Please feel free to add to them.


Depedencies
-----------

  o RPCAuth 0.4

    If you downloaded Quills as a tarball, the correct version of RPCAuth will be bundled with it.
    If not, you can check out RPCAuth from svn.plone.org/svn/collective/RPCAuth/trunk

  o Plone 2.0.5 or higher
