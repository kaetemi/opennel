INTRODUCTION:
-------------

Ligoscape is a landscape generation method based on the NeL landscape technologie.

PROJECT STRUCTURE:
------------------

The project is composed of :

* A static library for shared code
* A 3dsmax plugin to check, debug and export ligo data from max
* A MFC tool to paint the landscape with base zone

PROJECT DEPENDENCIES:
---------------------
